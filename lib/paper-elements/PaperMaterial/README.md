## Usage

Basic usage

```
{{> PaperMaterial}}
```

Add a custom className

```
{{> PaperMaterial className="my-custom-class"}}
```

Add a content

```
{{# PaperMaterial}}[content]{{# PaperMaterial}}
```

Change the elevation

```
{{> PaperMaterial elevation=3}}
```
